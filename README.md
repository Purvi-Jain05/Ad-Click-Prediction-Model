# Ad Click Prediction Model

## 📌 Overview
This project predicts whether a user will click on an advertisement using Machine Learning.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

## 📊 Dataset
Advertising dataset containing user behavior and demographics.

## 🚀 Model
Logistic Regression model used for prediction.
